# MobileSlicer 0.1.1 Beta

MobileSlicer 0.1.1 Beta is a small Android beta update focused on broader
device compatibility, cleaner release packaging, and more reliable printer
connection setup.

This is the recommended beta build for Android testers.

## What's New

- Improved compatibility with newer Android devices that require 16 KB native
  library alignment.
- Strengthened release packaging so the beta installs as a non-debuggable,
  release-signed Android app.
- Refined Bambu LAN setup validation to catch incomplete printer details before
  connection or upload attempts.
- Kept in-progress scanner work out of the public beta interface while it
  continues to mature.
- Updated the public download page to point directly to this beta release.

## Install

Download `MobileSlicer-0.1.1-beta-v4.apk` from the assets below, then open it on
your Android device. Android may ask you to allow installs from your browser or
file manager before continuing.

If you already have an older MobileSlicer beta installed, this build should
install as an update.

## Beta Notes

- Bambu LAN printing features are still beta. Connection setup now gives better
  validation, but printer behavior can still vary by model, firmware, and local
  network configuration.
- Thingiverse import work is included in the codebase, but account sign-in and
  direct import may depend on server-side availability during the beta.
- Please report install issues, crashes, printer connection problems, and model
  import problems through the project feedback channels.
