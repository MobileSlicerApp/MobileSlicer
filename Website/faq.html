FAQ

Beta questions.

Short answers for people trying the Android beta.

Is MobileSlicer on Google Play?

Not yet. The beta is currently distributed as an APK through the registration page.

Is MobileSlicer available for iOS?

MobileSlicer is currently only available for Android, but once the Android release is stable, I will start developing for iOS.

Is the beta free?

Yes. MobileSlicer is free, open source, and ad free.

Do I need to create an account?

No. The beta form only asks for your Android device, 3D printer, and what you want to test.

Why do you ask for my device and printer?

That information helps prioritize Android device coverage, printer profiles, and compatibility work.

Does slicing happen on-device?

Yes. The project is built around local Android UI with a wrapped native slicing engine.

Does MobileSlicer upload my models?

No. The beta workflow is built around local model loading, local slicing, local preview, and file export.

What Android version is required?

The app currently targets Android 8.0 or newer, which is API 26 and up.

Which model formats work right now?

The public beta flow is focused on STL and 3MF model workflows.

What printers are supported?

The beta focuses on curated profile coverage first. Registration asks for your printer so tester demand can guide compatibility work.

Can I use it on a tablet?

Yes. The interface is designed for phones and tablets, with larger touch targets and mobile-first layouts.

Can I send files directly to my printer?

Direct printer sending is not the first beta focus. The current workflow centers on slicing, previewing, and exporting or sharing generated files.

How do I update to a newer beta?

Download the latest beta from this site and install it over the previous build. Release notes are available on the releases page.

What should I test first?

Import, workspace controls, profile selection, slicing, G-code preview, export or share, and saved projects are the most useful beta areas to test.

Why does Android warn about installing the APK?

The beta is sideloaded, so Android may show an install warning. Only install builds downloaded from the official MobileSlicer domain.
